⚠️ **DO NOT REPORT SECURITY VULNERABILITIES PUBLICLY**

If you discover a security vulnerability, please:

1. **DO NOT** open a public issue
2. Email us directly at: security@anygle.app
3. Include details about the vulnerability
4. Allow 48 hours for initial response

**For non-sensitive security questions**, you can use this template:

**Describe the concern**
A clear and concise description of the security concern.

- [ ] Medium
- [ ] High
- [ ] Critical

**Additional context**
Any other context or information about the security concern.
